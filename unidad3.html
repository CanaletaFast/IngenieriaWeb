JavaScript Básico

En esta sección aprenderemos los fundamentos de JavaScript, un lenguaje de programación esencial para la interactividad en la web.

Ejemplo 1: Alertas

alert('¡Hola, mundo!');

Resultado:

¡Hola! Vamos a entender cómo funcionan las alertas en JavaScript.

¿Qué es una alerta?

Una alerta es una ventana emergente que muestra un mensaje al usuario. Es útil para notificaciones simples, advertencias o información importante.

Entendiendo el código

Este código muestra una alerta con el mensaje "¡Hola, mundo!" cuando se ejecuta.

Desglose del código

alert('¡Hola, mundo!');

Elementos clave:

  • alert(): Función incorporada de JavaScript que muestra una ventana emergente con un mensaje.
  • '¡Hola, mundo!': El mensaje que se muestra en la alerta.

Resultado visual

Al hacer clic en el botón "Mostrar Alerta", se mostrará una ventana emergente con el mensaje "¡Hola, mundo!".

Ejemplo 2: Variables y Tipos de Datos

let nombre = 'Ana';
const edad = 30;
var esEstudiante = true;

console.log(nombre);
console.log(edad);
console.log(esEstudiante);

Resultado:

¡Hola! Vamos a entender cómo declarar variables y manejar diferentes tipos de datos en JavaScript.

¿Qué son las variables?

Las variables son contenedores que almacenan datos. En JavaScript, se pueden declarar usando let, const o var.

Tipos de Datos

  • Cadenas de texto (Strings): Texto encerrado entre comillas. Ejemplo: 'Ana'.
  • Números (Numbers): Valores numéricos. Ejemplo: 30.
  • Booleanos (Booleans): Valores lógicos que pueden ser true o false.

Entendiendo el código

Este código declara tres variables con diferentes tipos de datos y luego imprime sus valores en la consola.

Desglose del código

let nombre = 'Ana';
const edad = 30;
var esEstudiante = true;

console.log(nombre);
console.log(edad);
console.log(esEstudiante);

Elementos clave:

  • let nombre = 'Ana';: Declara una variable llamada nombre con el valor 'Ana'.
  • const edad = 30;: Declara una constante llamada edad con el valor 30. Las constantes no pueden ser reasignadas.
  • var esEstudiante = true;: Declara una variable llamada esEstudiante con el valor true.
  • console.log(): Función que imprime mensajes en la consola del navegador.

Resultado visual

Al hacer clic en el botón "Mostrar Variables en la Consola", los valores de las variables se imprimirán en la consola del navegador.

Cómo Ver la Consola:

  1. Haz clic derecho en la página y selecciona "Inspeccionar" o "Inspect".
  2. Navega a la pestaña "Consola" o "Console".
  3. Verás los valores de las variables impresos allí.

Ejemplo 3: Funciones

function saludar(nombre) {
    return `¡Hola, ${nombre}!`;
}

let mensaje = saludar('Carlos');
console.log(mensaje);

Resultado:

¡Hola! Vamos a entender cómo crear y utilizar funciones en JavaScript.

¿Qué es una función?

Una función es un bloque de código reutilizable que realiza una tarea específica. Puedes pasarle datos (parámetros) y puede devolver un resultado.

Entendiendo el código

Este código define una función llamada saludar que toma un nombre como parámetro y devuelve un saludo personalizado. Luego, la función se invoca y su resultado se imprime en la consola.

Desglose del código

function saludar(nombre) {
    return `¡Hola, ${nombre}!`;
}

let mensaje = saludar('Carlos');
console.log(mensaje);

Elementos clave:

  • function saludar(nombre) { ... }: Declara una función llamada saludar que acepta un parámetro nombre.
  • return `¡Hola, ${nombre}!`;: Retorna una cadena de texto que incluye el nombre proporcionado.
  • let mensaje = saludar('Carlos');: Invoca la función saludar con el argumento 'Carlos' y almacena el resultado en la variable mensaje.
  • console.log(mensaje);: Imprime el mensaje en la consola.

Resultado visual

Al hacer clic en el botón "Saludar", se mostrará una alerta con el mensaje "¡Hola, Carlos!" y también se imprimirá en la consola.

Ejemplo 4: Eventos

function manejarClick() {
    alert('¡Botón clickeado!');
}

document.getElementById('miBoton').addEventListener('click', manejarClick);

Resultado:

¡Hola! Vamos a entender cómo manejar eventos en JavaScript.

¿Qué son los eventos?

Los eventos son acciones que ocurren en el navegador, como clics, movimientos del ratón, presiones de teclas, etc. JavaScript puede escuchar y responder a estos eventos para crear interactividad.

Entendiendo el código

Este código define una función llamada manejarClick que muestra una alerta cuando se hace clic en el botón con el ID "miBoton". Luego, se agrega un listener para el evento "click" que ejecuta esta función.

Desglose del código

function manejarClick() {
    alert('¡Botón clickeado!');
}

document.getElementById('miBoton').addEventListener('click', manejarClick);

Elementos clave:

  • function manejarClick() { ... }: Define una función llamada manejarClick que muestra una alerta.
  • document.getElementById('miBoton'): Selecciona el botón con el ID "miBoton".
  • .addEventListener('click', manejarClick): Añade un listener para el evento "click" que ejecuta la función manejarClick cuando se dispara el evento.

Resultado visual

Al hacer clic en el botón "Haz Clic Aquí", se mostrará una alerta con el mensaje "¡Botón clickeado!".

¿Qué es el DOM?

El DOM (Document Object Model) es una representación estructural de la página web que permite a JavaScript interactuar y modificar los elementos HTML y CSS de manera dinámica.

Entendiendo el código

Este código define una función llamada cambiarTexto que cambia el contenido de un párrafo cuando se hace clic en un botón. Luego, se agrega un listener para el evento "click" en el botón que ejecuta esta función.

Desglose del código

function cambiarTexto() {
    document.getElementById('texto').innerHTML = '¡El texto ha sido cambiado!';
}

document.getElementById('cambiarTexto').addEventListener('click', cambiarTexto);

Elementos clave:

  • function cambiarTexto() { ... }: Define una función que cambia el contenido de un párrafo.
  • document.getElementById('texto'): Selecciona el párrafo con el ID "texto".
  • .innerHTML = '¡El texto ha sido cambiado!';: Cambia el contenido HTML del párrafo.
  • document.getElementById('cambiarTexto'): Selecciona el botón con el ID "cambiarTexto".
  • .addEventListener('click', cambiarTexto): Añade un listener para el evento "click" que ejecuta la función cambiarTexto cuando se dispara el evento.

Resultado visual

Al hacer clic en el botón "Cambiar Texto", el contenido del párrafo cambiará de "Este es el texto original." a "¡El texto ha sido cambiado!".

Ejemplo 7: Arrays y Bucles

function mostrarFrutas() {
    let frutas = ['Manzana', 'Banana', 'Cereza'];
    for (let i = 0; i < frutas.length; i++) {
        console.log(`Fruta ${i + 1}: ${frutas[i]}`);
    }
}

Resultado:

¡Hola! Vamos a entender cómo manejar arrays y bucles en JavaScript.

¿Qué es un array?

Un array es una estructura de datos que almacena una colección de elementos. Cada elemento puede accederse mediante un índice.

¿Qué es un bucle?

Un bucle es una estructura que permite repetir una sección de código varias veces. En este ejemplo, utilizamos un bucle for.

Entendiendo el código

Este código define una función llamada mostrarFrutas que crea un array de frutas y luego utiliza un bucle para imprimir cada fruta en la consola con su número correspondiente.

Desglose del código

function mostrarFrutas() {
    let frutas = ['Manzana', 'Banana', 'Cereza'];
    for (let i = 0; i < frutas.length; i++) {
        console.log(`Fruta ${i + 1}: ${frutas[i]}`);
    }
}

Elementos clave:

  • let frutas = ['Manzana', 'Banana', 'Cereza'];: Declara un array llamado frutas con tres elementos.
  • for (let i = 0; i < frutas.length; i++) { ... }: Inicia un bucle que itera desde 0 hasta la longitud del array frutas.
  • console.log(`Fruta ${i + 1}: ${frutas[i]}`);: Imprime el número de la fruta y su nombre en la consola.

Resultado visual

Al hacer clic en el botón "Mostrar Frutas en la Consola", se imprimirán las siguientes líneas en la consola:

Fruta 1: Manzana
Fruta 2: Banana
Fruta 3: Cereza

  • Elemento Existente 2

¡Hola! Vamos a entender cómo crear y agregar elementos al DOM usando JavaScript.

¿Qué es la manipulación del DOM?

La manipulación del DOM permite a JavaScript crear, modificar y eliminar elementos en una página web de manera dinámica, mejorando la interactividad y la experiencia del usuario.

Entendiendo el código

Este código define una función llamada agregarElemento que crea un nuevo elemento de lista <li> con el texto "Nuevo Elemento" y lo agrega a una lista existente cada vez que se hace clic en un botón.

Desglose del código

function agregarElemento() {
    let lista = document.getElementById('miLista');
    let nuevoElemento = document.createElement('li');
    nuevoElemento.textContent = 'Nuevo Elemento';
    lista.appendChild(nuevoElemento);
}

document.getElementById('agregarElemento').addEventListener('click', agregarElemento);

Elementos clave:

  • function agregarElemento() { ... }: Define una función que crea y agrega un nuevo elemento a la lista.
  • let lista = document.getElementById('miLista');: Selecciona la lista con el ID "miLista".
  • let nuevoElemento = document.createElement('li');: Crea un nuevo elemento <li>.
  • nuevoElemento.textContent = 'Nuevo Elemento';: Establece el texto del nuevo elemento de lista.
  • lista.appendChild(nuevoElemento);: Añade el nuevo elemento de lista al final de la lista existente.
  • document.getElementById('agregarElemento').addEventListener('click', agregarElemento);: Añade un listener para el evento "click" que ejecuta la función agregarElemento cuando se dispara el evento.

Resultado visual

Al hacer clic en el botón "Agregar Elemento", se añadirá un nuevo elemento de lista con el texto "Nuevo Elemento" a la lista existente.

Ejemplo 10: Fetch API para Obtener Datos

function obtenerDatos() {
    fetch('https://jsonplaceholder.typicode.com/posts/1')
        .then(response => response.json())
        .then(data => {
            console.log('Datos recibidos:', data);
            document.getElementById('datos').innerHTML = JSON.stringify(data, null, 2);
        })
        .catch(error => console.error('Error:', error));
}

document.getElementById('obtenerDatos').addEventListener('click', obtenerDatos);

Resultado:


                        

                    

                    

                        
¡Hola! Vamos a entender cómo utilizar la Fetch API para obtener datos de una API externa.


                        
¿Qué es la Fetch API?

                        
La Fetch API es una interfaz moderna que permite hacer solicitudes HTTP desde JavaScript para obtener recursos de manera asíncrona.


                        
Entendiendo el código

                        
Este código define una función llamada obtenerDatos que realiza una solicitud GET a una API externa, procesa la respuesta y muestra los datos en la consola y en la página web.


                        
Desglose del código

                        
function obtenerDatos() {
    fetch('https://jsonplaceholder.typicode.com/posts/1')
        .then(response => response.json())
        .then(data => {
            console.log('Datos recibidos:', data);
            document.getElementById('datos').innerHTML = JSON.stringify(data, null, 2);
        })
        .catch(error => console.error('Error:', error));
}

document.getElementById('obtenerDatos').addEventListener('click', obtenerDatos);


                        
Elementos clave:

                        
    
                            
  • function obtenerDatos() { ... }: Define una función que realiza una solicitud a una API y maneja la respuesta.
    • 
                            
  • fetch('https://jsonplaceholder.typicode.com/posts/1'): Realiza una solicitud GET a la URL proporcionada.
    • 
                            
  • .then(response => response.json()): Procesa la respuesta y la convierte en un objeto JSON.
    • 
                            
  • .then(data => { ... }): Maneja los datos recibidos, imprimiéndolos en la consola y mostrando su contenido en el elemento con el ID "datos".
    • 
                            
  • .catch(error => console.error('Error:', error)): Captura y maneja cualquier error que ocurra durante la solicitud.
    • 
                            
  • document.getElementById('obtenerDatos').addEventListener('click', obtenerDatos);: Añade un listener para el evento "click" que ejecuta la función obtenerDatos cuando se dispara el evento.
    • 
                        


                        
Resultado visual

                        
Al hacer clic en el botón "Obtener Datos", se realizará una solicitud a la API. Los datos recibidos se imprimirán en la consola y se mostrarán en el área designada en la página web.


                        
Nota: En este ejemplo, he utilizado la API pública JSONPlaceholder para demostrar cómo obtener datos. Puedes reemplazar la URL con la de tu propia API si lo deseas.

¿Por qué crear elementos con JavaScript?

                        
Crear y agregar elementos al DOM permite dinamizar el contenido de una página web, añadir nuevos elementos basados en acciones del usuario, o cargar contenido de manera asíncrona.


                        
Entendiendo el código
