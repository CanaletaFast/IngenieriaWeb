PHP Básico

En esta sección aprenderemos los fundamentos de PHP, un lenguaje de programación esencial para el desarrollo web del lado del servidor.

¿Qué es PHP?

PHP (Hypertext Preprocessor) es un lenguaje de programación de código abierto ampliamente utilizado para el desarrollo web del lado del servidor. Permite generar contenido dinámico, interactuar con bases de datos y manejar formularios, entre otras funcionalidades.

Entendiendo el código

Este es un script básico de PHP que imprime el mensaje "¡Hola, mundo!" en el navegador.

Desglose del código

<?php
echo "¡Hola, mundo!";
?>

Elementos clave:

  • <?php ... ?>: Etiquetas de apertura y cierre de PHP que indican dónde comienza y termina el código PHP.
  • echo "¡Hola, mundo!";: Imprime el texto "¡Hola, mundo!" en el navegador.

Resultado visual

Al ejecutar este script en un servidor PHP, verás el texto "¡Hola, mundo!" en tu navegador.

¿Qué son las variables?

Las variables son contenedores que almacenan información. En PHP, se declaran con el símbolo $ seguido del nombre de la variable.

  • Cadenas de texto (Strings): Texto encerrado entre comillas. Ejemplo: "Ana".

  • Booleanos (Booleans): Valores lógicos que pueden ser true o false.

¿Qué son las estructuras de control?

Las estructuras de control permiten dirigir el flujo de ejecución del programa. Las más comunes son las condicionales (if, elseif, else) y los bucles (for, while, foreach).

Entendiendo el código

Este script demuestra el uso de condicionales y bucles en PHP:

Condicionales:

Evalúa si una variable cumple ciertas condiciones y ejecuta bloques de código en consecuencia.

Bucle for:

Repite un bloque de código un número específico de veces.

Bucle while:

Repite un bloque de código mientras una condición sea verdadera.

Desglose del código

<?php
$numero = 10;

if ($numero > 0) {
    echo "$numero es positivo.<br>";
} elseif ($numero < 0) {
    echo "$numero es negativo.<br>";
} else {
    echo "$numero es cero.<br>";
}

// Bucle for
for ($i = 1; $i <= 5; $i++) {
    echo "Número: $i<br>";
}

// Bucle while
$j = 1;
while ($j <= 3) {
    echo "Contador: $j<br>";
    $j++;
}
?>

Elementos clave:

  • if ($numero > 0) { ... }: Verifica si $numero es mayor que 0.
  • elseif ($numero < 0) { ... }: Verifica si $numero es menor que 0.
  • else { ... }: Ejecuta este bloque si ninguna de las condiciones anteriores es verdadera.
  • for ($i = 1; $i <= 5; $i++) { ... }: Inicia un bucle que se repite 5 veces.
  • while ($j <= 3) { ... }: Inicia un bucle que se repite mientras $j sea menor o igual a 3.

¿Qué son los métodos GET y POST?

Son dos métodos HTTP que se utilizan para enviar datos desde un formulario HTML al servidor. GET envía los datos a través de la URL, mientras que POST los envía en el cuerpo de la solicitud, lo que es más seguro para datos sensibles.

Entendiendo el código

Este ejemplo consiste en dos archivos:

  1. formulario.php: Contiene el formulario HTML donde el usuario puede ingresar su nombre y email.
  2. procesar_formulario.php: Procesa los datos enviados desde el formulario y los muestra en el navegador.

Desglose del código

<form action="procesar_formulario.php" method="post">
    
    <input type="text" id="nombre" name="nombre" required><br><br>
    
    
    <input type="email" id="email" name="email" required><br><br>
    
    <input type="submit" value="Enviar">
</form>
  • action="procesar_formulario.php": Especifica el archivo que procesará los datos del formulario.
  • method="post": Indica que se utilizará el método POST para enviar los datos.
  • name="nombre" y name="email": Nombres de las variables que se enviarán al servidor.
  • required: Atributo que hace que el campo sea obligatorio.

Seguridad: Utilizamos la función htmlspecialchars() para evitar ataques de Cross-Site Scripting (XSS) al escapar caracteres especiales.

Ejemplo 8: CRUD – Crear, Leer, Actualizar, Eliminar Datos

<?php
$servername = "localhost";
$username = "root";
$password = "";
$dbname = "mi_basedatos";

// Crear conexión
$conn = new mysqli($servername, $username, $password, $dbname);

// Verificar conexión
if ($conn->connect_error) {
    die("Conexión fallida: " . $conn->connect_error);
}

// Crear tabla
$sqlCrearTabla = "CREATE TABLE IF NOT EXISTS Usuarios (
    id INT(6) UNSIGNED AUTO_INCREMENT PRIMARY KEY,
    nombre VARCHAR(30) NOT NULL,
    email VARCHAR(50),
    reg_date TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP
)";

if ($conn->query($sqlCrearTabla) === TRUE) {
    echo "Tabla Usuarios creada o ya existe.<br>";
} else {
    echo "Error creando la tabla: " . $conn->error . "<br>";
}

// Insertar datos
$sqlInsertar = "INSERT INTO Usuarios (nombre, email) VALUES ('Laura', 'laura@example.com')";

if ($conn->query($sqlInsertar) === TRUE) {
    echo "Nuevo registro creado exitosamente.<br>";
} else {
    echo "Error: " . $sqlInsertar . "<br>" . $conn->error . "<br>";
}

// Leer datos
$sqlLeer = "SELECT id, nombre, email FROM Usuarios";
$resultado = $conn->query($sqlLeer);

if ($resultado->num_rows > 0) {
    echo "Usuarios:<br>";
    while($fila = $resultado->fetch_assoc()) {
        echo "ID: " . $fila["id"]. " - Nombre: " . $fila["nombre"]. " - Email: " . $fila["email"]. "<br>";
    }
} else {
    echo "0 resultados.<br>";
}

// Actualizar datos
$sqlActualizar = "UPDATE Usuarios SET email='laura.nuevo@example.com' WHERE nombre='Laura'";

if ($conn->query($sqlActualizar) === TRUE) {
    echo "Registro actualizado exitosamente.<br>";
} else {
    echo "Error actualizando registro: " . $conn->error . "<br>";
}

// Eliminar datos
$sqlEliminar = "DELETE FROM Usuarios WHERE nombre='Laura'";

if ($conn->query($sqlEliminar) === TRUE) {
    echo "Registro eliminado exitosamente.<br>";
} else {
    echo "Error eliminando registro: " . $conn->error . "<br>";
}

// Cerrar conexión
$conn->close();
?>

Resultado:

Para ver el resultado de este código:

  1. Asegúrate de tener una base de datos llamada mi_basedatos en tu servidor MySQL.
  2. Crea un archivo llamado crud.php con el contenido del código anterior.
  3. Ejecuta el archivo en tu servidor PHP.

Salida Esperada:

Tabla Usuarios creada o ya existe.
Nuevo registro creado exitosamente.
Usuarios:
ID: 1 - Nombre: Laura - Email: laura@example.com
Registro actualizado exitosamente.
Registro eliminado exitosamente.

Ejemplo 9: Manejo de Sesiones y Cookies en PHP

<?php
// Iniciar sesión
session_start();

// Establecer una variable de sesión
$_SESSION["usuario"] = "Juan";

// Establecer una cookie que expira en 30 días
setcookie("preferencia", "dark_mode", time() + (86400 * 30), "/");

// Verificar si la sesión y la cookie están establecidas
if(isset($_SESSION["usuario"])) {
    echo "Usuario en sesión: " . $_SESSION["usuario"] . "<br>";
} else {
    echo "No hay usuario en sesión.<br>";
}

if(isset($_COOKIE["preferencia"])) {
    echo "Preferencia de usuario: " . $_COOKIE["preferencia"] . "<br>";
} else {
    echo "No hay preferencias establecidas.<br>";
}
?>

Resultado:

Para ver el resultado de este código:

  1. Crea un archivo llamado sesiones_cookies.php con el contenido del código anterior.
  2. Ejecuta el archivo en tu servidor PHP.
  3. Verifica que se muestran los valores de la sesión y la cookie en tu navegador.
  4. Recarga la página para asegurarte de que las sesiones y las cookies persisten.

Salida Esperada:

Usuario en sesión: Juan
Preferencia de usuario: dark_mode

¡Hola! Vamos a entender cómo manejar sesiones y cookies en PHP.

¿Qué son las sesiones?

Las sesiones permiten almacenar información en el servidor para ser utilizada en múltiples páginas durante la navegación del usuario.

¿Qué son las cookies?

Las cookies son pequeños archivos almacenados en el navegador del usuario que permiten guardar información sobre el mismo.

Entendiendo el código

Este script realiza las siguientes acciones:

  1. Inicia una sesión.
  2. Establece una variable de sesión llamada usuario.
  3. Establece una cookie llamada preferencia que expira en 30 días.
  4. Verifica y muestra los valores de la sesión y la cookie.

Desglose del código

<?php
// Iniciar sesión
session_start();

// Establecer una variable de sesión
$_SESSION["usuario"] = "Juan";

// Establecer una cookie que expira en 30 días

Elementos clave:

  • session_start();: Inicia una sesión o reanuda una existente.
  • $_SESSION["usuario"] = "Juan";: Establece una variable de sesión.

<?php

$username = "root";
$password = "";

// Crear tabla Categorias si no existe
$sqlCrearCategorias = "CREATE TABLE IF NOT EXISTS Categorias (
    id INT(6) UNSIGNED AUTO_INCREMENT PRIMARY KEY,
    nombre VARCHAR(30) NOT NULL
)";

// Insertar categorías
$sqlInsertarCategoria = "INSERT INTO Categorias (nombre) VALUES ('Electrónica'), ('Ropa'), ('Hogar')";

    id INT(6) UNSIGNED AUTO_INCREMENT PRIMARY KEY,
    nombre VARCHAR(30) NOT NULL,
    precio DECIMAL(10,2),
    categoria_id INT(6) UNSIGNED,
    reg_date TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
    FOREIGN KEY (categoria_id) REFERENCES Categorias(id)
)";

    ('Smartphone', 599.99, 1),
