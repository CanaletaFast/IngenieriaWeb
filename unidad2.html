HTML Básico

En esta sección aprenderemos los elementos fundamentales de HTML.

Ejemplo 1: Estructura básica de un documento HTML

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <title>Mi Primera Página</title>
</head>
<body>
    <!-- Contenido va aquí -->
</body>
</html>

Resultado:

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <title>Mi Primera Página</title>
</head>
<body>
    <!-- Contenido va aquí -->
</body>
</html>

¡Hola! Vamos a desglosar este ejemplo para entender la estructura básica de un documento HTML.

¿Qué es HTML?

HTML (HyperText Markup Language) es el lenguaje estándar para crear y estructurar páginas web. Utiliza etiquetas para definir elementos como párrafos, encabezados, enlaces, imágenes, etc.

Entendiendo el código

Este es un ejemplo de la estructura mínima necesaria para crear una página web con HTML.

Desglose del código

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <title>Mi Primera Página</title>
</head>
<body>
    <!-- Contenido va aquí -->
</body>
</html>

Elementos clave:

  • <!DOCTYPE html>: Declara que el documento es un documento HTML5.
  • <html lang="es">: Define el inicio del documento HTML y establece el idioma principal como español.
  • <head>: Contiene metadatos sobre el documento, como el juego de caracteres y el título.
  • <meta charset="UTF-8">: Especifica la codificación de caracteres utilizada, en este caso, UTF-8.
  • <title>Mi Primera Página</title>: Define el título de la página que se muestra en la pestaña del navegador.
  • <body>: Contiene el contenido visible de la página web, como texto, imágenes, enlaces, etc.
  • <!-- Contenido va aquí -->: Un comentario que indica dónde se debe agregar el contenido principal.

Resultado visual

Este código por sí solo no muestra contenido visible en la página, ya que está vacío dentro del <body>. Sin embargo, proporciona la estructura básica necesaria para empezar a construir una página web.

¿Qué son los títulos y párrafos?

Los títulos se utilizan para definir la jerarquía y estructura del contenido, mientras que los párrafos contienen el texto principal.

Entendiendo el código

Este ejemplo muestra cómo crear un título principal y un párrafo de texto en HTML.

Desglose del código

<h1>Título Principal</h1>
<p>Este es un párrafo de texto.</p>

Elementos clave:

  • <h1>: Define un encabezado de nivel 1, el más importante en la jerarquía de títulos.
  • <p>: Define un párrafo de texto.

Resultado visual

Al renderizar este código en un navegador, verás un título grande y destacado seguido de un párrafo de texto.

¿Qué son las listas en HTML?

Las listas permiten organizar elementos de manera estructurada. Hay dos tipos principales:

  • Listas No Ordenadas (<ul>): Utilizan viñetas para listar elementos sin un orden específico.
  • Listas Ordenadas (<ol>): Utilizan números para listar elementos en un orden específico.

Elementos clave:

  • <ul>: Define una lista no ordenada.
  • <ol>: Define una lista ordenada.
  • <li>: Define un elemento de lista.

Resultado visual

  • Lista No Ordenada: Muestra tres elementos con viñetas.
  • Lista Ordenada: Muestra tres elementos numerados.

Ejemplo 4: Enlaces y Imágenes

<a href="https://www.ejemplo.com">Visita Ejemplo.com</a>
<img src="imagen.jpg" alt="Descripción de la imagen">

Resultado:

Visita Ejemplo.com

¡Hola! Vamos a entender cómo crear enlaces e insertar imágenes en HTML.

¿Qué son los enlaces y las imágenes?

Los enlaces permiten a los usuarios navegar a otras páginas o sitios web, mientras que las imágenes enriquecen el contenido visual de la página.

Entendiendo el código

Este ejemplo muestra cómo crear un enlace que redirige a una página externa y cómo insertar una imagen en tu página web.

Desglose del código

<a href="https://www.ejemplo.com">Visita Ejemplo.com</a>
<img src="imagen.jpg" alt="Descripción de la imagen">

Elementos clave:

  • <a href="https://www.ejemplo.com">Visita Ejemplo.com</a>: Crea un enlace que lleva al usuario a ejemplo.com. El texto "Visita Ejemplo.com" es lo que verá y podrá hacer clic el usuario.
  • <img src="imagen.jpg" alt="Descripción de la imagen">: Inserta una imagen en la página.
    • src="imagen.jpg": Especifica la ruta de la imagen que se va a mostrar. Asegúrate de que la imagen esté en la ubicación correcta o ajusta la ruta según corresponda.
    • alt="Descripción de la imagen": Proporciona una descripción alternativa para la imagen, útil para accesibilidad y SEO.
    • width="200": Define el ancho de la imagen en píxeles.

Resultado visual

  • Enlace: Un texto clicable que lleva a los usuarios a ejemplo.com.
  • Imagen: Una imagen mostrada con un ancho de 200 píxeles. Asegúrate de tener una imagen llamada "imagen.jpg" en la ubicación especificada o ajusta la ruta del atributo src.

Consejo: Siempre utiliza el atributo alt en las imágenes para mejorar la accesibilidad y proporcionar contexto cuando la imagen no se puede cargar.

¡Hola! Vamos a entender cómo crear tablas en HTML.

¿Qué es una tabla en HTML?

Las tablas permiten organizar datos en filas y columnas, facilitando la presentación de información tabular.

Entendiendo el código

Este ejemplo muestra cómo crear una tabla con dos columnas: "Nombre" y "Edad", y tres filas de datos.

Desglose del código

<table border="1">
    <tr>
        <th>Nombre</th>
        <th>Edad</th>
    </tr>
    <tr>
        <td>Juan</td>
        <td>25</td>
    </tr>

  • <tr>: Define una fila en la tabla.
  • <th>: Define una celda de encabezado en una tabla. Por defecto, el texto está en negrita y centrado.
  • <td>: Define una celda de datos en una tabla.

Resultado visual

La tabla mostrará dos columnas: "Nombre" y "Edad", con tres filas de datos para Juan y Ana.

Consejo: Evita utilizar el atributo border para estilizar las tablas. En su lugar, utiliza CSS para una mayor flexibilidad y mejores prácticas de diseño.

<form action="/submit" method="post">
    <label for="nombre">Nombre:</label>
    <input type="text" id="nombre" name="nombre"><br><br>

    <label for="email">Correo Electrónico:</label>
    <input type="email" id="email" name="email"><br><br>

    <input type="submit" value="Enviar">
</form>

Resultado:





¡Hola! Vamos a entender cómo crear formularios en HTML.

¿Qué es un formulario en HTML?

Los formularios permiten a los usuarios ingresar datos que se pueden enviar a un servidor para su procesamiento. Son esenciales para funcionalidades como registros, inicios de sesión, búsquedas, y más.

Entendiendo el código

Este ejemplo muestra un formulario básico con campos para el nombre y el correo electrónico, y un botón de envío.

Desglose del código

<form action="/submit" method="post">
    <label for="nombre">Nombre:</label>
    <input type="text" id="nombre" name="nombre"><br><br>

    <label for="email">Correo Electrónico:</label>
    <input type="email" id="email" name="email"><br><br>

    <input type="submit" value="Enviar">
</form>

Elementos clave:

  • <form action="/submit" method="post">: Define un formulario que enviará los datos a la URL "/submit" utilizando el método POST.
  • <label for="nombre">Nombre:</label>: Etiqueta para el campo de entrada de nombre. El atributo for está vinculado al id del campo de entrada correspondiente.
  • <input type="text" id="nombre" name="nombre">: Campo de entrada de texto para el nombre del usuario.
  • <br><br>: Saltos de línea para espaciar los elementos. (Nota: Es mejor utilizar CSS para el espaciado en lugar de saltos de línea).
  • <input type="email" id="email" name="email">: Campo de entrada para el correo electrónico, con validación básica de formato de email.
  • <input type="submit" value="Enviar">: Botón para enviar el formulario.

Resultado visual

El formulario mostrará dos campos de entrada: uno para el nombre y otro para el correo electrónico, seguido de un botón "Enviar". Al hacer clic en "Enviar", los datos se enviarán al servidor especificado en el atributo action.

Consejos:

  • Usa CSS para el estilo: Evita usar etiquetas <br> para el espaciado. En su lugar, utiliza CSS para controlar el diseño y el espacio entre los elementos.
  • Validación: Implementa validación tanto en el lado del cliente (con JavaScript) como en el lado del servidor para asegurar que los datos ingresados son correctos y seguros.
  • Accesibilidad: Utiliza etiquetas <label> adecuadamente para mejorar la accesibilidad y la experiencia del usuario.

<div class="contenedor">
    <h2>Título dentro de un div</h2>
    <p>Este es un párrafo dentro de un div.</p>
</div>

<span style="color: blue;">Texto dentro de un span</span>

Resultado:

Título dentro de un div

Este es un párrafo dentro de un div.

Texto dentro de un span

¡Hola! Vamos a entender cómo funcionan las etiquetas <div> y <span> en HTML.

¿Qué son <div> y <span>?

<div> y <span> son etiquetas contenedoras utilizadas para agrupar y estilizar otros elementos en HTML.

  • <div>: Un contenedor de bloque que se utiliza para agrupar elementos y aplicarles estilos o para estructurar la página.
  • <span>: Un contenedor en línea utilizado para agrupar partes de texto u otros elementos en línea para aplicar estilos específicos.

Entendiendo el código

Este ejemplo muestra cómo utilizar <div> para agrupar elementos de bloque y <span> para agrupar elementos en línea.

Desglose del código

<div class="contenedor">
    <h2>Título dentro de un div</h2>
    <p>Este es un párrafo dentro de un div.</p>
</div>

<span style="color: blue;">Texto dentro de un span</span>

Elementos clave:

  • <div class="contenedor">: Crea un contenedor de bloque con la clase "contenedor". Esto permite aplicar estilos específicos a todos los elementos dentro de este <div>.
  • <h2>Título dentro de un div</h2>: Un título de nivel 2 dentro del <div>.
  • <p>Este es un párrafo dentro de un div.</p>: Un párrafo de texto dentro del <div>.
  • <span style="color: blue;">Texto dentro de un span</span>: Un texto en línea con estilo aplicado directamente que cambia su color a azul.

Resultado visual

  • <div>: Agrupa el título y el párrafo, permitiendo aplicar estilos o estructuras específicas al conjunto.
  • <span>: Muestra un texto en línea con el color azul aplicado, sin afectar a otros elementos en la misma línea.

Consejos:

  • Uso adecuado: Utiliza <div> para agrupar elementos de bloque y <span> para elementos en línea.
  • Estilos CSS: Es recomendable aplicar estilos mediante clases o IDs en lugar de estilos en línea para mantener el HTML limpio y facilitar el mantenimiento.
  • Semántica: Aunque <div> y <span> son útiles para estructurar y estilizar, considera utilizar etiquetas semánticas como <header>, <nav>, <main>, etc., para mejorar la accesibilidad y la SEO.

<p>Este párrafo está visible.</p>
<!-- 
    Este es un párrafo comentado y no será visible.
    <p>No se mostrará en la página.</p>
-->

Resultado:

¡Hola! Vamos a entender cómo funcionan los comentarios en HTML.

¿Qué son los comentarios en HTML?

Los comentarios son notas que puedes agregar en tu código HTML para explicar o recordar cosas. No se muestran en el navegador y no afectan la visualización de la página.

Entendiendo el código

Este ejemplo muestra cómo agregar comentarios en HTML y cómo afectan la visualización de la página.

Desglose del código

<!-- Este es un comentario en HTML -->
<p>Este párrafo está visible.</p>
<!-- 
    Este es un párrafo comentado y no será visible.
    <p>No se mostrará en la página.</p>
-->

Elementos clave:

  • <!-- Este es un comentario en HTML -->: Un comentario sencillo que explica algo en el código.
  • <p>Este párrafo está visible.</p>: Un párrafo que se muestra en la página.
  • <!-- Este es un párrafo comentado y no será visible. <p>No se mostrará en la página.</p> -->: Un comentario multilínea que incluye una etiqueta de párrafo que no se mostrará.

Resultado visual

Solo el párrafo "Este párrafo está visible." se mostrará en la página. El párrafo dentro del comentario no será visible.

Consejos:

  • Usa comentarios para: Explicar secciones del código, recordar tareas pendientes, o deshabilitar temporalmente partes del código sin eliminarlas.
  • No abuses de los comentarios: Evita llenar tu HTML de comentarios innecesarios, ya que puede dificultar la lectura del código.

    <title>Página con Meta Tags</title>
</head>

Resultado:

Revisa el título de la pestaña y la responsividad de la página.

¡Hola! Vamos a entender cómo funcionan las Meta Tags en HTML.

¿Qué son las Meta Tags?

Las Meta Tags proporcionan información sobre el documento HTML que no se muestra directamente en la página. Son esenciales para la optimización de motores de búsqueda (SEO), la compatibilidad con navegadores y la adaptabilidad a diferentes dispositivos.

Entendiendo el código

Este ejemplo muestra cómo definir las Meta Tags para la codificación de caracteres y la configuración de la viewport para diseño responsivo.

Desglose del código

<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<title>Página con Meta Tags</title>

Elementos clave:

  • <meta charset="UTF-8">: Especifica la codificación de caracteres utilizada en el documento, asegurando que se muestren correctamente los caracteres especiales.
  • <meta name="viewport" content="width=device-width, initial-scale=1.0">: Controla cómo se escala y se muestra la página en dispositivos móviles. Hace que la página sea responsiva.
  • <title>Página con Meta Tags</title>: Define el título de la página que se muestra en la pestaña del navegador.

Resultado visual

Las Meta Tags no son visibles directamente en la página, pero afectan aspectos clave:

  • Título: Aparece en la pestaña del navegador.
  • Responsividad: La página se adaptará al tamaño de pantalla del dispositivo, asegurando una buena experiencia en móviles y tabletas.

Consejos:

  • Usa Meta Tags correctamente: Asegúrate de que las Meta Tags estén bien definidas para mejorar la compatibilidad y la optimización de tu página.
  • SEO: Utiliza Meta Tags adicionales como <meta name="description"> para proporcionar descripciones que ayuden a los motores de búsqueda a entender el contenido de tu página.

¡Hola! Vamos a entender cómo usar elementos semánticos en HTML para estructurar mejor tu página.

¿Qué son los elementos semánticos en HTML?

Los elementos semánticos proporcionan significado al contenido, describiendo claramente su propósito tanto para los desarrolladores como para los navegadores y motores de búsqueda. Mejora la accesibilidad y el SEO de tu página.

Entendiendo el código

Este ejemplo muestra cómo utilizar elementos semánticos como <header>, <nav>, <main>, <section> y <footer> para estructurar una página web.

Desglose del código

<header>
    <h1>Encabezado de la Página</h1>
</header>

<nav>
    <ul>
        <li><a href="#inicio">Inicio</a></li>
        <li><a href="#servicios">Servicios</a></li>
        <li><a href="#contacto">Contacto</a></li>
    </ul>
</nav>

<main>
    <section>
        <h2>Contenido Principal</h2>
        <p>Este es el contenido principal de la página.</p>
    </section>
</main>

<footer>
    <p>Pie de página con información de contacto.</p>
</footer>

Elementos clave:

  • <header>: Define el encabezado de la página, que generalmente contiene el título y otros elementos introductorios.
  • <nav>: Define una sección de navegación con enlaces a otras partes de la página o a páginas externas.
  • <main>: Define el contenido principal de la página, único por página.
  • <section>: Define una sección temática dentro del contenido principal.
  • <footer>: Define el pie de página, que suele contener información de contacto, derechos de autor, etc.

Resultado visual

  • Header: Muestra el título de la página.
  • Nav: Presenta una lista de enlaces de navegación.
  • Main: Contiene la sección principal con un título y un párrafo.
  • Footer: Muestra información de contacto y derechos de autor.

Consejos:

  • Usa elementos semánticos: Facilita la comprensión de la estructura de la página tanto para humanos como para máquinas.
  • Organiza tu contenido: Utiliza <section> para agrupar contenido relacionado, mejorando la legibilidad y la mantenibilidad del código.
  • Mejora la accesibilidad: Los elementos semánticos ayudan a tecnologías asistivas a interpretar correctamente el contenido.

CSS Básico

En esta sección exploraremos cómo aplicar estilos a nuestra página HTML utilizando CSS.

Ejemplo 1: Estilos en línea

<p style="color: blue;">Este texto es azul.</p>

Resultado:

Este texto es azul.

¡Hola! Entiendo que este código de CSS pueda parecer un poco confuso al principio. Te lo explicaré de forma sencilla.

¿Qué es CSS?

CSS (Cascading Style Sheets) es un lenguaje de diseño que se utiliza para dar estilo a las páginas web. Con CSS, podemos definir cómo se verán los elementos de una página, como el color del texto, el tamaño de las fuentes, el espaciado, etc.

Entendiendo el código

Este código utiliza **estilos en línea** para aplicar un color específico a un párrafo.

Desglose del código

<p style="color: blue;">Este texto es azul.</p>

Propiedades CSS:

  • color: blue;: Cambia el color del texto del párrafo a azul.

Resultado visual

Cuando apliques este estilo a un elemento <p> en tu HTML, verás que el texto dentro del párrafo se mostrará en color azul.

Ejemplo 2: Estilos internos

<head>
    <style>
        p {
            font-size: 18px;
            color: green;
        }
    </style>
</head>
<body>
    <p>Este párrafo tiene estilos internos.</p>
</body>

Resultado:

Este párrafo tiene estilos internos.

¡Hola! Vamos a desglosar este ejemplo para que entiendas mejor cómo funcionan los estilos internos en CSS.

¿Qué son los estilos internos?

Los estilos internos se definen dentro de la etiqueta <style> en la sección <head> de tu documento HTML. Estos estilos afectan a todo el documento a menos que se especifiquen de otra manera.

Entendiendo el código

Este código aplica estilos a todos los elementos <p> (párrafos) dentro del documento.

Desglose del código

<style>
    p {
        font-size: 18px;
        color: green;
    }
</style>

Propiedades CSS:

  • font-size: 18px;: Establece el tamaño de la fuente del texto a 18 píxeles.
  • color: green;: Cambia el color del texto a verde.

Resultado visual

Al aplicar este estilo, cualquier párrafo <p> en tu HTML mostrará su texto con un tamaño de fuente de 18 píxeles y en color verde.

Ejemplo 3: Estilos externos

En el archivo style.css:

p {
    color: red;
    text-align: center;
}

En el archivo HTML:

<head>
    <link rel="stylesheet" href="style.css">
</head>
<body>
    <p>Este párrafo utiliza estilos externos.</p>
</body>

Resultado:

Este párrafo utiliza estilos externos.

¡Hola! Vamos a entender cómo funcionan los estilos externos en CSS.

¿Qué son los estilos externos?

Los estilos externos se definen en un archivo separado con extensión .css y se vinculan a tu documento HTML mediante la etiqueta <link>. Esto permite mantener el HTML limpio y facilita la reutilización de estilos en múltiples páginas.

Entendiendo el código

En este ejemplo, estamos aplicando estilos a los párrafos <p> utilizando un archivo CSS externo.

Desglose del código

/* style.css */
p {
    color: red;
    text-align: center;
}

Propiedades CSS:

  • color: red;: Cambia el color del texto a rojo.
  • text-align: center;: Centra el texto dentro del párrafo.

Enlace en el HTML:

<link rel="stylesheet" href="style.css">

Esta línea de código en la sección <head> vincula el archivo style.css al documento HTML, aplicando los estilos definidos allí.

Resultado visual

¿Qué es CSS?

CSS (Cascading Style Sheets) es un lenguaje de diseño que se utiliza para dar estilo a las páginas web. Con CSS, podemos definir cómo se verán los elementos de una página, como el color del texto, el tamaño de las fuentes, el espaciado, etc.

Entendiendo el código

Este código CSS está definiendo estilos para dos tipos de elementos: aquellos que tienen una clase específica y aquellos que tienen un ID específico.

Clases y IDs

  • Clases: Son como etiquetas que puedes asignar a múltiples elementos en tu HTML. Puedes pensar en ellas como grupos de elementos que quieres que tengan el mismo estilo. Se declaran con un punto (.) seguido del nombre de la clase.
  • IDs: Son identificadores únicos que se asignan a un solo elemento en tu HTML. Se utilizan cuando quieres aplicar un estilo muy específico a un elemento en particular. Se declaran con un signo de número (#) seguido del nombre del ID.

Desglose del código

/* Estilos para elementos con la clase 'destacado' */
.destacado {
    background-color: yellow;
    font-weight: bold;
}

/* Estilos para el elemento con el ID 'principal' */
#principal {
    border: 2px solid #4CAF50;
    padding: 10px;
}

Uso del código

  • .destacado: Esta regla de estilo se aplica a cualquier elemento HTML que tenga la clase "destacado" asignada. A esos elementos se les dará un fondo amarillo y el texto se mostrará en negrita.
  • #principal: Esta regla de estilo se aplica únicamente al elemento HTML que tenga el ID "principal". A ese elemento se le añadirá un borde sólido verde de 2 píxeles y un relleno interno de 10 píxeles.

¿Qué significa "Este párrafo tiene la clase 'destacado'. Este div tiene el ID 'principal'."?

Esto significa que en el código HTML correspondiente, hay un párrafo (etiqueta <p>) al que se le ha asignado la clase "destacado" y un contenedor (etiqueta <div>) al que se le ha asignado el ID "principal".

En resumen:

  • Clases: Se utilizan para aplicar estilos a múltiples elementos.
  • IDs: Se utilizan para aplicar estilos a un único elemento.

El código CSS define cómo se verán los elementos que tienen una clase o un ID específico. El HTML asigna las clases y los IDs a los elementos.

Entendiendo el código

Este código CSS utiliza **pseudo-clases** para aplicar estilos a elementos en situaciones específicas.

Desglose del código

/* Cambia el color del enlace al pasar el cursor */
a:hover {
    color: orange;
}

/* Estilos para el primer hijo de una lista */
ul li:first-child {

Primera parte: a:hover { color: orange; }

  • color: orange;: Cambia el color del texto del enlace a naranja cuando se cumple la condición anterior (es decir, cuando el cursor está sobre el enlace).

Segunda parte: ul li:first-child { font-weight: bold; }

  • :first-child: Esta pseudo-clase selecciona el primer elemento hijo de un elemento padre. En este caso, el primer elemento <li> dentro de cada <ul>.
  • font-weight: bold;: Hace que el texto del primer elemento de cada lista se muestre en negrita.

En resumen: Esta parte del código pone en negrita el primer elemento de cada lista no ordenada.

  • Listas: El primer elemento de cada lista no ordenada se mostrará en negrita.

Ejemplo 6: Box Model

/* Margen, padding y borde */
.box {
    margin: 20px;
    padding: 15px;
    border: 2px solid #333;
    background-color: #e0e0e0;
}

Resultado:

Este div demuestra el modelo de caja con margen, padding y borde.

¡Claro! Vamos a analizar el código CSS y explicar qué hace cada parte:

Propiedades CSS:

  • margin: 20px;: Establece un margen de 20 píxeles alrededor del elemento. El margen crea un espacio vacío entre el elemento y los elementos adyacentes.
  • padding: 15px;: Establece un relleno interno de 15 píxeles dentro del elemento. El relleno crea un espacio entre el borde del elemento y su contenido.
  • border: 2px solid #333;: Aplica un borde sólido de 2 píxeles al elemento. El color del borde es #333 (un gris oscuro).
  • background-color: #e0e0e0;: Establece el color de fondo del elemento a un gris claro (#e0e0e0).

Resultado visual

Cuando apliques este estilo a un elemento con la clase "box" en tu HTML, verás un cuadro con las siguientes características:

  • Espacio alrededor: Un margen de 20 píxeles separará el cuadro de otros elementos.
  • Espacio interno: El contenido del cuadro estará a 15 píxeles de distancia del borde.
  • Borde: El cuadro tendrá un borde sólido de 2 píxeles de color gris oscuro.
  • Fondo: El fondo del cuadro será de color gris claro.

Ejemplo de HTML:

<div class="box">
    Este div demuestra el modelo de caja con margen, padding y borde.
</div>

Al combinar este HTML con el CSS anterior, verás el resultado visual descrito.

Concepto del Modelo de Caja (Box Model)

Este ejemplo demuestra el concepto fundamental del modelo de caja en CSS. Cada elemento en una página web se puede considerar como una caja, y esta caja tiene cuatro áreas principales:

  • Contenido: El texto o imágenes dentro del elemento.
  • Padding: El espacio entre el contenido y el borde.
  • Borde: La línea que rodea el elemento.
  • Margen: El espacio entre el borde del elemento y otros elementos.

El modelo de caja es esencial para entender cómo se posicionan y espacián los elementos en una página web.

¿Qué es Flexbox?

Flexbox (Flexible Box Layout) es un modelo de diseño en CSS que facilita la alineación y distribución de elementos dentro de un contenedor, incluso cuando su tamaño es desconocido o dinámico.

Entendiendo el código

Este código define un contenedor flexible y tres elementos dentro de él que se ajustan automáticamente.

Desglose del código

/* Contenedor Flex */
.flex-container {
    display: flex;
    justify-content: space-around;
    align-items: center;
    background-color: #f2f2f2;
    padding: 20px;
}

/* Elementos Flex */
.flex-item {
    background-color: #4CAF50;
    color: white;
    padding: 20px;
    margin: 10px;
    flex: 1;
    text-align: center;
}

Propiedades CSS del Contenedor Flex:

  • display: flex;: Define al elemento como un contenedor flex, habilitando el modelo Flexbox.
  • justify-content: space-around;: Distribuye los elementos hijos con espacios iguales alrededor de ellos.
  • align-items: center;: Alinea los elementos hijos verticalmente en el centro del contenedor.
  • background-color: #f2f2f2;: Establece el color de fondo del contenedor.
  • padding: 20px;: Añade un relleno interno de 20 píxeles al contenedor.

Propiedades CSS de los Elementos Flex:

  • background-color: #4CAF50;: Establece el color de fondo de los elementos a un verde.
  • color: white;: Cambia el color del texto a blanco.
  • padding: 20px;: Añade un relleno interno de 20 píxeles a cada elemento.
  • margin: 10px;: Añade un margen externo de 10 píxeles alrededor de cada elemento.
  • flex: 1;: Permite que los elementos crezcan y se encogen según el espacio disponible, ocupando un espacio igual entre ellos.
  • text-align: center;: Centra el texto dentro de cada elemento.

Resultado visual

Al aplicar este estilo, verás un contenedor con tres elementos flexibles distribuidos equitativamente con espacios alrededor. Cada elemento tiene un fondo verde, texto blanco, y se ajusta automáticamente al tamaño del contenedor.

¿Qué son las Media Queries?

Las Media Queries son una característica de CSS que permite aplicar estilos específicos según las características del dispositivo, como el ancho de la pantalla. Son fundamentales para crear diseños responsivos que se adapten a diferentes tamaños de pantalla.

/* Estilos por defecto */

    padding: 20px;
    text-align: center;
}

/* Estilos para pantallas menores a 600px */
@media (max-width: 600px) {
    .box-responsive {
        background-color: lightcoral;
        padding: 10px;
    }
}

Propiedades CSS:

  • background-color: lightblue;: Establece el color de fondo de la caja a azul claro.
  • padding: 20px;: Añade un relleno interno de 20 píxeles.
  • text-align: center;: Centra el texto dentro de la caja.

Media Query:

@media (max-width: 600px) {
    .box-responsive {
        background-color: lightcoral;
        padding: 10px;
    }
}
  • @media (max-width: 600px): Define una Media Query que se aplica cuando el ancho de la pantalla es de 600 píxeles o menos.
  • background-color: lightcoral;: Cambia el color de fondo de la caja a coral claro.
  • padding: 10px;: Reduce el relleno interno a 10 píxeles.

Esto asegura que la caja se vea bien y sea legible tanto en dispositivos de escritorio como en móviles.

¿Qué son las transiciones en CSS?

Las transiciones en CSS permiten cambiar propiedades CSS de manera gradual durante un período de tiempo especificado. Esto crea efectos visuales suaves cuando un usuario interactúa con un elemento, como al pasar el cursor por encima.

Entendiendo el código

Este código define un botón que cambia de color de fondo cuando el usuario pasa el cursor por encima, utilizando una transición suave.

Desglose del código

/* Botón con transición */
.boton-transicion {
    background-color: #008CBA;
    color: white;
    padding: 15px 32px;
    text-align: center;
    text-decoration: none;
    display: inline-block;
    transition: background-color 0.5s ease;
    border: none;
    border-radius: 4px;
    cursor: pointer;
}

.boton-transicion:hover {
    background-color: #005f6a;
}

Propiedades CSS:

  • background-color: #008CBA;: Establece el color de fondo inicial del botón a un azul.
  • color: white;: Cambia el color del texto a blanco.
  • padding: 15px 32px;: Añade un relleno interno de 15 píxeles arriba y abajo, y 32 píxeles a los lados.
  • text-align: center;: Centra el texto dentro del botón.
  • text-decoration: none;: Elimina cualquier subrayado del texto.
  • display: inline-block;: Permite que el botón tenga dimensiones y pueda alinearse con otros elementos.
  • transition: background-color 0.5s ease;: Aplica una transición de 0.5 segundos al cambiar el color de fondo, con una función de tiempo "ease" para suavizar el efecto.
  • border: none;: Elimina cualquier borde predeterminado.
  • border-radius: 4px;: Redondea las esquinas del botón con un radio de 4 píxeles.
  • cursor: pointer;: Cambia el cursor a una mano cuando se pasa por encima del botón, indicando que es clicable.

Hover State:

.boton-transicion:hover {
    background-color: #005f6a;
}
  • background-color: #005f6a;: Cambia el color de fondo del botón a un azul más oscuro cuando el cursor pasa por encima.

Resultado visual

Al pasar el cursor sobre el botón, notarás que el color de fondo cambia de #008CBA a #005f6a de manera suave y gradual gracias a la transición definida. Este efecto mejora la experiencia del usuario al proporcionar retroalimentación visual interactiva.

¿Qué son las animaciones en CSS?

Las animaciones en CSS permiten crear movimientos y transiciones complejas aplicando cambios en las propiedades CSS de un elemento a lo largo del tiempo. Utilizan la regla @keyframes para definir los estados de inicio y fin de la animación, y luego aplican esa animación a un elemento específico.

Entendiendo el código

Este código define una animación llamada "desvanecer" que cambia la opacidad de un elemento de 0 a 1, creando un efecto de desvanecimiento al cargarse.

Desglose del código

/* Animación de desvanecimiento */
@keyframes desvanecer {
    from { opacity: 0; }
    to { opacity: 1; }
}

.animacion-desvanecer {
    animation: desvanecer 2s ease-in-out;
}

Definición de la animación:

@keyframes desvanecer {
    from { opacity: 0; }
    to { opacity: 1; }
}
  • @keyframes desvanecer: Define una animación llamada "desvanecer".
  • from { opacity: 0; }: Estado inicial de la animación, con opacidad 0 (completamente transparente).
  • to { opacity: 1; }: Estado final de la animación, con opacidad 1 (completamente opaco).

Aplicación de la animación:

.animacion-desvanecer {
    animation: desvanecer 2s ease-in-out;
}
  • animation: desvanecer 2s ease-in-out;: Aplica la animación "desvanecer" al elemento. La animación dura 2 segundos y utiliza una función de tiempo "ease-in-out" para suavizar el inicio y el final de la animación.

Resultado visual

Al cargar la página, el texto dentro del elemento con la clase animacion-desvanecer comenzará completamente transparente y se desvanecerá hasta ser completamente visible en 2 segundos. Este efecto puede mejorar la apariencia y la experiencia de usuario al introducir elementos de manera suave en la página.
