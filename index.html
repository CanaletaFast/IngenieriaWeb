Unidades Didácticas

  1. Unidad Didáctica 1: Introducción y Contexto

    En esta unidad se presentan los objetivos del curso y se define el proyecto que desarrollarán los estudiantes.

    • Reflexión sobre ideas para el proyecto web.
    • Definición de la idea del proyecto a desarrollar.

  2. Unidad Didáctica 2: Entornos de Desarrollo, HTML y CSS [+]

    En esta unidad se instala y configura el entorno de desarrollo y se aprenden los fundamentos de HTML y CSS.

    • Instalación de herramientas: Visual Studio Code, XAMPP, Git y GitHub.
    • Creación de la estructura básica del proyecto con HTML y CSS.

    Ejemplo de código HTML:

    <!-- Estructura básica de un documento HTML5 -->
<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <title>Título de la página</title>
</head>
<body>
    <h1>Hola Mundo</h1>
</body>
</html>

    Ejemplo de código CSS:

    /* Estilos para el cuerpo de la página */
body {
    background-color: #f0f0f0;
    color: #333;
}

/* Estilos para el título */
h1 {
    font-size: 2em;
    color: #4CAF50;
}

  3. Unidad Didáctica 3: JavaScript y el DOM [+]

    En esta unidad se introducen los conceptos básicos de JavaScript y cómo manipular el DOM.

    • Introducción a la sintaxis de JavaScript.
    • Manipulación del DOM y eventos.
    • Validaciones y eventos en JavaScript.

    Ejemplo de código JavaScript:

    // Seleccionar un elemento del DOM
const boton = document.getElementById('miBoton');

// Añadir un evento al botón
boton.addEventListener('click', function() {
    alert('¡Has hecho clic en el botón!');
});

  4. Unidad Didáctica 4: Back-End con PHP y MySQL [+]

    En esta unidad se aprende a desarrollar el lado del servidor con PHP y a utilizar bases de datos MySQL.

    • Fundamentos de PHP.
    • Configuración y uso de bases de datos MySQL.
    • Procesamiento de formularios y seguridad en PHP.

    Ejemplo de código PHP:

    <?php
// Conexión a la base de datos MySQL
$servername = "localhost";
$username = "usuario";
$password = "contraseña";
$dbname = "mi_base_de_datos";

// Crear conexión
$conn = new mysqli($servername, $username, $password, $dbname);

// Verificar conexión
if ($conn->connect_error) {
    die("Conexión fallida: " . $conn->connect_error);
}

// Consulta SQL para insertar datos
$sql = "INSERT INTO usuarios (nombre, email) VALUES ('Juan', 'juan@example.com')";

if ($conn->query($sql) === TRUE) {
    echo "Nuevo registro creado exitosamente";
} else {
    echo "Error: " . $sql . "<br>" . $conn->error;
}

$conn->close();
?>

  5. Unidad Didáctica 5: Profundización en PHP y Introducción a Node.js [+]

    En esta unidad se profundiza en PHP con programación orientada a objetos y se introduce Node.js.

    • Programación Orientada a Objetos en PHP.
    • Introducción a Node.js y su aplicación en el proyecto.

    Ejemplo de código PHP con Programación Orientada a Objetos:

    <?php
// Definición de la clase Usuario
class Usuario {
    private $nombre;
    private $email;

    // Constructor de la clase
    public function __construct($nombre, $email) {
        $this->nombre = $nombre;
        $this->email = $email;
    }

    // Método para guardar el usuario
    public function guardar() {
        // Código para guardar el usuario en la base de datos
    }

    // Método para obtener el nombre del usuario
    public function obtenerNombre() {
        return $this->nombre;
    }
}

// Crear una instancia de la clase Usuario
$usuario = new Usuario('Ana', 'ana@example.com');

// Imprimir el nombre del usuario
echo $usuario->obtenerNombre();
?>

    Ejemplo de código Node.js:

    // Importar el módulo http
const http = require('http');

// Crear el servidor
const server = http.createServer((req, res) => {
    res.statusCode = 200;
    res.setHeader('Content-Type', 'text/plain');
    res.end('Hola desde Node.js\n');
});

// Definir el puerto y host
const port = 3000;
const host = 'localhost';

// Iniciar el servidor
server.listen(port, host, () => {
    console.log(`Servidor corriendo en http://${host}:${port}/`);
});

  6. Unidad Didáctica 6: Diseño Responsivo, Intercambio de Datos y AJAX [+]

    En esta unidad se mejoran las técnicas de diseño responsivo y se trabaja con AJAX para comunicación asíncrona.

    • Diseño responsivo avanzado.
    • Formatos de intercambio de datos: XML, JSON y YAML.
    • Uso de Fetch API para comunicación asíncrona.

    Ejemplo de código JavaScript utilizando Fetch API:

    // Realizar una solicitud GET a una API
fetch('https://api.example.com/datos')
    .then(response => response.json())
    .then(data => {
        // Procesar los datos recibidos
        console.log(data);
    })
    .catch(error => {
        console.error('Error:', error);
    });
